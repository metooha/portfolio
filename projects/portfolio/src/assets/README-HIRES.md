# High-resolution (2x) assets

To improve sharpness on retina displays, add **@2x** versions of images and place them in this folder (`src/assets`). The app will use them automatically on high-DPI screens.

## How to export higher resolution from Figma

1. In Figma, select the frame or layer you want to export.
2. In the right panel, open the **Export** section (or use the export icon at the bottom of the Design panel).
3. Click **+** to add an export setting.
4. Choose **PNG** (or **JPG** if you prefer).
5. Set scale to **2x** or **3x** (2x is usually enough for retina).
6. Export and save the file with the exact name listed below for that asset.
